# Artificial Intelligence and Machine Learning Work

In addition to backend systems engineering, I have worked extensively with applied AI and machine learning to automate complex workflows.

These projects typically involved integrating machine learning models into production systems where they could operate reliably at scale.

Some of the key AI-driven systems I worked on include:

### Fraud Detection Systems

I built fraud detection models that analyze financial transaction patterns to identify suspicious activity. These systems combine machine learning models with backend services that process transaction data and flag anomalies.

### OCR-Based Document Processing

I developed systems that use **Optical Character Recognition (OCR)** to automate document processing workflows. These pipelines extract structured information from scanned documents, reducing the need for manual data entry.

### NLP Systems for Healthcare Data

I implemented **BERT-based natural language processing models** designed to extract medical entities from healthcare-related documents. These systems help transform unstructured medical text into structured information that can be processed by other systems.

### Image Classification Pipelines

I also worked on **image classification systems** used in healthcare workflows. These models analyze image data and classify content to assist downstream processing systems.

Building these AI pipelines required integrating machine learning models with backend services, data pipelines, and cloud infrastructure.
